# Debug Panel 옵션 가이드

개발 모드(`npm run dev`) 또는 디버그 빌드(`npm run build:debug`)에서 표시되는 파티클 튜닝 패널.

## Global Settings

### Particle Size
- **기본값**: 0.06 | **범위**: 0.01 ~ 0.15
- 각 파티클 점(dot)의 기본 크기. 모든 모델에 일괄 적용.

### Near Size (×)
- **기본값**: 0.1 | **범위**: 0.1 ~ 5.0
- 카메라에 **가까운** 파티클의 크기 배율.
- 0.1 = 앞쪽 파티클이 매우 작게 → 뒤쪽이 상대적으로 부각되는 깊이감 연출.
- 1.0 = 기본 크기 유지.

### Far Size (×)
- **기본값**: 2.9 | **범위**: 0.1 ~ 5.0
- 카메라에서 **먼** 파티클의 크기 배율.
- 2.9 = 뒤쪽 파티클이 크게 보여 역원근 깊이감 강조.
- Near Size와 조합하여 파티클 클라우드의 입체감 제어.

### Dome Radius
- **기본값**: 0.7 | **범위**: 0.05 ~ 2.0
- 마우스 효과(scatter, orbit, size)가 적용되는 원형 영역의 반경.
- 로컬 좌표 기준이며, 오브젝트 스케일에 맞게 자동 보정됨 (`mouseRadius / userScale`).
- Show Dome Area를 켜면 빨간 원으로 시각화 가능.

### Activation Radius
- **기본값**: 7.0 | **범위**: 1.0 ~ 10.0
- 마우스가 오브젝트 중심에서 이 거리(월드 유닛) 이내일 때만 돔 효과 계산 시작.
- 성능 최적화용 외곽 차단 영역. 이 밖에서는 파티클 루프를 건너뜀.
- Dome Radius보다 충분히 크게 설정해야 돔 효과가 정상 동작.

### Show Dome Area
- **기본값**: off
- 돔 반경을 빨간 원(ring)으로 시각화. 효과 영역 확인 시 유용.

### Scatter Strength
- **기본값**: 1.2 | **범위**: 0.1 ~ 3.0
- 마우스 진입/이동 시 파티클이 바깥으로 밀려나는 강도.
- 높을수록 더 크게 흩어짐. 마우스 velocity에 비례하여 자동 조절됨.
- 마우스 정지(호버링) 시 0으로 감쇠 → 파티클 제자리 복귀.

### Parallax
- **기본값**: 0.08 | **범위**: 0 ~ 0.3
- 마우스 스크린 위치에 따른 오브젝트 미세 회전 (라디안).
- 0.08 ≈ 4.5° 최대 기울기. 입체감/3D 깊이 효과 부여.
- 0 = 패럴랙스 비활성.

## Size Effect

마우스 돔 영역 내 파티클의 크기를 확대하는 효과.

### Enable
- **기본값**: on
- 사이즈 확대 효과 on/off 토글.

### Strength
- **기본값**: 0.5 | **범위**: 0 ~ 2.0
- 최대 확대 배율. 0.5 = 돔 중심 파티클이 최대 1.5x.
- 마우스 이동 시 전체 강도 적용, 호버링 시 30%만 적용 (미세 볼록 효과).

---

## Orbit

마우스 돔 영역 내 파티클이 접선 방향으로 진동하는 공전 효과.

### Speed
- **기본값**: 6.0 | **범위**: 0 ~ 6.0
- 공전 속도 (rad/s). 높을수록 빠르게 진동.
- 파티클마다 랜덤 위상(phase)이 부여되어 유기적으로 보임.

### Strength
- **기본값**: 1.0 | **범위**: 0 ~ 1.0
- 공전 진폭(반경). 높을수록 접선 방향 이동 거리 증가.
- 마우스 velocity에 비례하여 자동 조절. 호버링 시 0으로 감쇠.

---

## Spring Physics

파티클 오프셋 복귀 방식을 lerp 대신 스프링 물리로 전환.

### Enable Spring
- **기본값**: off
- off = 선형 보간(lerp)으로 부드럽게 이동/복귀.
- on = 스프링 기반으로 탄성 있는 바운스 동작.

### Stiffness
- **기본값**: 180 | **범위**: 20 ~ 500
- 스프링 강성 계수. 높을수록 빠르게 목표 위치로 수렴.

### Damping
- **기본값**: 12 | **범위**: 1 ~ 30
- 스프링 감쇠 계수. 낮을수록 바운스가 오래 지속.
- 높으면 빠르게 안정, 낮으면 탄성 진동.

## Per-Model Settings (예: 0: Light Bulb)

모델별로 개별 조정 가능한 값. 로드된 모델 수만큼 폴더 생성.

### Scale
- **기본값**: 1.0 | **범위**: 0.1 ~ 3.0
- 모델별 추가 크기 조절. 최종 크기 = `8유닛 정규화 × config.scale × 이 값`.

### Rotation X / Y / Z
- **기본값**: 0 | **범위**: -π ~ π
- 모델의 기본 회전각 (라디안). Parallax 회전과 합산됨.

### Particles (max N)
- **범위**: 100 ~ 최대 로드 수
- 표시할 파티클 수. 줄이면 성능 향상, 늘리면 디테일 증가.

### Set
- 현재 Scale, Rotation, Particles 값을 **localStorage에 저장**.
- 새로고침 후에도 유지됨. 코드 기본값을 변경하려면 Export Config 사용.

---

## Export Config

모든 모델의 현재 설정을 콘솔에 `sceneConfig.ts` 형식으로 출력.
튜닝 완료 후 코드에 반영할 때 사용.

---

## 내부 동작: mouseActivity

디버그 패널에는 노출되지 않지만 핵심 내부 변수.

- **마우스 진입 시**: `mouseActivity = 1.0` (burst)
- **마우스 이동 중**: velocity에 비례하여 0~1 유지
- **마우스 정지(호버링)**: 0.02 rate로 서서히 0으로 감쇠 (ease-out)

Scatter, Orbit, Size Effect의 동적 부분은 모두 `mouseActivity`에 비례.
호버링 시에는 Size Effect의 30%만 유지 (미세 볼록).
